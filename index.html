About

저는 새로운 내용을 배울 때 예제가 실행되는 것만 확인하지 않고, 코드를 직접 바꾸거나 이상한 값을 넣어보며 어떤 오류가 나는지 확인합니다. 이 방식 덕분에 프로그램이 어떤 흐름으로 동작하는지 빠르게 이해하고, 문제가 생겼을 때 원인을 남들보다 먼저 짚어내는 편입니다.

세운 목표는 계획을 쪼개서 꾸준히 실행하는 편입니다. 1학년 1학기에는 학과 수업과 병행하며 SQLD와 ADsP를 동시에 준비해 취득했습니다. 학교 프로젝트로 시작한 보안 앱 Security Hub는 과제로 끝내지 않고 자료를 더 조사해 한국정보통신학회 춘계종합학술대회에서 발표까지 마쳤습니다.

사람들과 협업하며 문제를 함께 풀어가는 과정을 좋아하고, 배운 것을 남에게 설명할 때 가장 큰 보람을 느낍니다.
